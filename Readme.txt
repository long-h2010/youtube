Hướng dẫn chạy project:
1. Truy cập https://www.apachefriends.org/download.html tải và cài đặt XAMPP cho hệ điều hành đang dùng.
2. Mở XAMPP Control Panel, chọn Start cho Module Apache và MySQL.
3. Sau khi 2 Module đã chạy thành công, chọn Admin ở Module MySQL.
4. Trang web phpMyAdmin sẽ hiện lên, chọn import và chọn file database.sql cung cấp cùng thư mục source.
5. Truy cập https://nodejs.org/en/download tải và cài đặt Node cho hệ điều hành đang dùng.
6. Mở command line có trên máy và đi tới đường dẫn */source.
7. Chạy dòng lệnh npm install hoặc yarn install (Nếu máy có cài đặt yarn).
8. Chạy dòng lệnh npm start hoặc yarn start.
9. Truy cập localhost:3000, đường dẫn hiện trên command line.

Có thể thay đổi lại cài đặt kết nối database bằng cách:
- Mở tập tin */source/utils/credentials.js
- Thay đổi host, user, password tương ứng.
- Lưu và chạy lại.
